# Texture attribution

Planet surface maps are **Solar System Scope** textures (CC BY 4.0), hosted on [Wikimedia Commons](https://commons.wikimedia.org/wiki/Category:Solar_System_Scope).

- Source project: [Solar System Scope](https://www.solarsystemscope.com/textures/)
- License: [Creative Commons Attribution 4.0](https://creativecommons.org/licenses/by/4.0/)
- Credit: © Solar System Scope / Wikimedia Commons contributors

8K bundles in this folder were downloaded for same-origin WebGL use in Solar Planets. 2K files remain as fallbacks where 8K is unavailable.

NASA and public-domain space imagery principles apply; these maps are widely used for educational 3D solar system visualizations.
